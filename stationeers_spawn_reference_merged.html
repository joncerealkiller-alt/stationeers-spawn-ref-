Stationeers Vanilla Spawn XML Reference

Interactive offline reference for the built-in spawn XML system. This focuses on vanilla-safe root fields, nested nodes, actions, and confirmed conditions that the base game already understands without requiring a runtime mod on the receiving side.

Confirmed action names from ThingSpawnData.Actions:

Logic BuildState Interaction MovePlayer Gene Quantity Percent Reagents Charge Gas SourceCode

Conditions are separate from actions. For example, <BuildState Index="-1" /> is an action, while <BuildState IsCompleted="true" /> is a condition.
